# 设计评审与校验

本文描述 在领域模型设计完成之后，在项目的概关键角色人员对，设计的进行回顾评审与校验，以便于确认大家都对需求，设计都达成一致

## 操作步骤

- 业务场景的文档评审
- 领域模型文档的评审

- 业务域设计文档的评审
- 应用的文档评审
